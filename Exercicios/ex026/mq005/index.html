Testando media queries

As media queries são uma funcionalidade poderosa do CSS que permitem a criação de layouts responsivos.
Com elas, é possível aplicar estilos diferentes dependendo do tamanho da tela do dispositivo, orientação, resolução e outras características.
Isso garante que o conteúdo do site seja exibido de forma adequada em diversos dispositivos, como smartphones, tablets e desktops.

Para utilizar media queries, é necessário definir uma regra CSS que especifique as condições em que os estilos devem ser aplicados.
Essas condições podem incluir a largura mínima ou máxima da tela, a orientação do dispositivo (paisagem ou retrato), entre outras.

Um exemplo simples de media query é o seguinte:
@media (max-width: 600px) { ... }.
Nesse caso, os estilos definidos dentro das chaves serão aplicados apenas se a largura da tela for igual ou inferior a 600 pixels.

Além de largura e orientação, as media queries também podem ser usadas para detectar outras características, como a resolução da tela.
Isso é útil para garantir que imagens e outros elementos gráficos sejam exibidos corretamente em dispositivos com diferentes densidades de pixels.

Em resumo, as media queries são uma ferramenta essencial para o desenvolvimento de sites modernos e responsivos.
Elas permitem que os desenvolvedores criem experiências de usuário adaptáveis, garantindo que o conteúdo seja acessível e visualmente agradável em qualquer dispositivo.
